## 1. Reflection API Object Model (Class → Field / Method / Constructor)

```
  Every loaded Java class is represented by a java.lang.Class<?> object.
  That Class object is the entry point to ALL reflection.

  ┌─────────────────────────────────────────────────────────────────────┐
  │                      java.lang.Class<T>                             │
  │                                                                     │
  │  Identity:                                                          │
  │    getName()              → "com.example.UserService"               │
  │    getSimpleName()        → "UserService"                           │
  │    getPackageName()       → "com.example"                           │
  │    getSuperclass()        → Class<Object> (or specific superclass)  │
  │    getInterfaces()        → Class<?>[]                              │
  │    getModifiers()         → int (PUBLIC, FINAL, ABSTRACT ...)       │
  │    isInterface()          → boolean                                 │
  │    isEnum()               → boolean                                 │
  │    isRecord()             → boolean                                 │
  │    isAnnotation()         → boolean                                 │
  │    getModule()            → Module (Java 9+)                        │
  └──────┬──────────────────────────────────────────────────────────────┘
         │
         ├──────────────────────────────────────────────────────────────
         │  FIELDS
         ├──→  getDeclaredFields()  → Field[]   (all in THIS class, incl. private)
         ├──→  getFields()          → Field[]   (public fields, incl. inherited)
         │     │
         │     └─── java.lang.reflect.Field
         │              getName()         → "userName"
         │              getType()         → Class<String>
         │              getModifiers()    → PRIVATE | FINAL
         │              getGenericType()  → ParameterizedType (for generics)
         │              get(Object obj)   → Object (reads field value)
         │              set(Object, val)  → void (writes field value)
         │              setAccessible(true)  → bypasses access control
         │
         ├──────────────────────────────────────────────────────────────
         │  METHODS
         ├──→  getDeclaredMethods()  → Method[]   (all in THIS class, incl. private)
         ├──→  getMethods()          → Method[]   (public, incl. inherited)
         │     │
         │     └─── java.lang.reflect.Method
         │              getName()            → "findById"
         │              getReturnType()      → Class<User>
         │              getParameterTypes()  → Class<?>[] {long.class}
         │              getExceptionTypes()  → Class<?>[]
         │              getAnnotations()     → Annotation[]
         │              invoke(Object, args) → Object (calls the method)
         │              isDefault()          → boolean (interface default method)
         │              isBridge()           → boolean (compiler-generated bridge)
         │
         ├──────────────────────────────────────────────────────────────
         │  CONSTRUCTORS
         └──→  getDeclaredConstructors()  → Constructor<?>[]
               getConstructors()          → Constructor<?>[]  (public only)
               │
               └─── java.lang.reflect.Constructor<T>
                        getParameterTypes()  → Class<?>[]
                        newInstance(args)    → T (creates an instance)
                        setAccessible(true)  → for non-public constructors

  ─────────────────────────────────────────────────────────────────────
  ALSO AVAILABLE:

  clazz.getAnnotations()                → all annotations on the class
  clazz.getAnnotation(SomeAnnotation.class) → specific annotation or null
  clazz.getDeclaredClasses()            → inner/nested classes
  clazz.getEnumConstants()              → T[] if class is an enum
  clazz.getRecordComponents()           → RecordComponent[] if record (Java 16+)
```

## 2. Dynamic Proxy Architecture

```
  CLIENT                        PROXY (generated at runtime)              TARGET
  ──────                        ────────────────────────────              ──────

  MyService ref;
  // ref is actually a $Proxy0 instance

  ref.doWork("arg1")
        │
        │  (ref is actually the Proxy object)
        ▼
  ┌─────────────────────────────────────────┐
  │  $Proxy0 (generated by JVM)             │
  │  implements: MyService                  │
  │                                         │
  │  public Object doWork(Object arg) {     │
  │      return h.invoke(                   │
  │          this,                          │──────────────────────────────────┐
  │          doWorkMethod,  // Method obj   │                                  │
  │          new Object[]{"arg1"}           │                                  │
  │      );                                 │                                  │
  │  }                                      │                                  │
  └─────────────────────────────────────────┘                                  │
                                                                               ▼
                                             ┌─────────────────────────────────────────┐
                                             │  YourInvocationHandler                  │
                                             │  implements InvocationHandler           │
                                             │                                         │
                                             │  invoke(proxy, method, args) {          │
                                             │    // BEFORE                            │
                                             │    log("before: " + method.getName());  │
                                             │                                         │
                                             │    Object result =                      │
                                             │      method.invoke(target, args); ──────┼──→ MyServiceImpl.doWork("arg1")
                                             │                                         │ ←── returns result
                                             │    // AFTER                             │
                                             │    log("after: " + method.getName());   │
                                             │                                         │
                                             │    return result;                       │
                                             │  }                                      │
                                             └─────────────────────────────────────────┘

  ─────────────────────────────────────────────────────────────────────────────────────

  HOW TO CREATE:

  ClassLoader   cl  = MyService.class.getClassLoader();
  Class<?>[]  ifaces = { MyService.class };               // MUST be interfaces
  InvocationHandler h = new YourInvocationHandler(target);

  MyService proxy = (MyService) Proxy.newProxyInstance(cl, ifaces, h);

  ─────────────────────────────────────────────────────────────────────────────────────

  CONSTRAINTS:
  ┌────────────────────────────────────────────────────────────────────────┐
  │  JDK Proxy → can only proxy INTERFACES                                 │
  │  CGLIB     → subclasses concrete classes; cannot proxy final classes   │
  │  ByteBuddy → most flexible; used by Mockito for mocking                │
  └────────────────────────────────────────────────────────────────────────┘
```

## 3. setAccessible — Bypassing Access Control

```
  NORMAL ACCESS CONTROL (compile time + runtime):

  ┌──────────────────────────────────────────────────────────────────┐
  │  class BankAccount {                                              │
  │      private double balance = 1000.0;  // private                │
  │  }                                                               │
  └──────────────────────────────────────────────────────────────────┘
                     │
  External code      │  acc.balance   → COMPILE ERROR
  attempts access    │  (also blocked at bytecode level by JVM
                     │   access checks in invokevirtual/getfield)

  ─────────────────────────────────────────────────────────────────────

  WITH setAccessible(true) — bypassing the gate:

  Field f = BankAccount.class.getDeclaredField("balance");

  [ACCESS CHECK GATE]
       │
       ├─ Is caller module = target module? → ALLOW
       ├─ Is package opened to caller? → ALLOW
       ├─ setAccessible(true) called? → ALLOW (bypass the gate)
       └─ Otherwise → InaccessibleObjectException (Java 9+)
                      IllegalAccessException   (Java 8 and earlier)

  f.setAccessible(true);   // disables per-call access checks
  f.get(acc);              // direct memory read, no check

  ─────────────────────────────────────────────────────────────────────

  JAVA 9+ MODULE LAYER (additional barrier):

  Unnamed module (classpath)    Named module (module-info.java)
  ─────────────────────────────────────────────────────────────────
  setAccessible(true) works     setAccessible(true) THROWS
  for most JDK classes          InaccessibleObjectException
  (with --add-opens flag)       UNLESS module declares:
                                  opens com.pkg to caller.module;
                                  or: opens com.pkg;  (to all)

  Timeline:
  Java 8:  setAccessible always works (no module system)
  Java 9-16: works but prints "illegal access" warning by default
  Java 17+:  strict encapsulation — InaccessibleObjectException
              without explicit opens
```

## 4. Class.forName() vs ClassLoader.loadClass()

```
  Class.forName("com.example.MyClass")
       │
       ├─ Finds class using the calling class's ClassLoader
       ├─ Loads the class (if not already loaded)
       ├─ RUNS STATIC INITIALIZERS immediately  ← key difference
       └─ Returns Class<?> object

  ClassLoader.loadClass("com.example.MyClass")
       │
       ├─ You specify which ClassLoader to use
       ├─ Loads the class (if not already loaded)
       ├─ Does NOT run static initializers ← key difference
       └─ Returns Class<?> object

  ─────────────────────────────────────────────────────────────────────

  WHEN STATIC INITIALIZERS MATTER:

  class JdbcDriver {
      static {
          // This runs when class is first INITIALIZED (not just loaded)
          DriverManager.registerDriver(new JdbcDriver());
          System.out.println("Driver registered!");
      }
  }

  Class.forName("com.mysql.Driver")    → static block RUNS → driver registered ✓
  ClassLoader.loadClass("com.mysql.Driver") → static block does NOT run → driver NOT registered ✗

  ─────────────────────────────────────────────────────────────────────

  THREE-ARGUMENT form of Class.forName():

  Class.forName(String name, boolean initialize, ClassLoader loader)
                                     ↑                    ↑
                              true = run static     which ClassLoader to use
                              false = don't run     (null = Bootstrap)

  // Load with custom ClassLoader, defer initialization
  Class<?> cls = Class.forName(
      "com.plugin.Service",
      false,                         // don't run static initializers yet
      pluginClassLoader              // use plugin's loader, not app's
  );
  // Later, force initialization:
  // cls.getDeclaredMethod("init").invoke(null);

  ─────────────────────────────────────────────────────────────────────

  LOADING vs LINKING vs INITIALIZATION:

  Load   → Read .class bytes, create Class object in JVM
  Link   → Verify bytecode, allocate static fields, resolve symbolic refs
  Init   → Run static blocks and assign static field initializers
                                                       ↑
                                         Class.forName() triggers this
                                         ClassLoader.loadClass() does NOT
```

## 5. Annotation Inspection at Runtime Flow

```
  SOURCE CODE:

  @Service                           ← @Retention(RUNTIME), @Target(TYPE)
  @Transactional(timeout = 30)       ← @Retention(RUNTIME), @Target(TYPE, METHOD)
  public class OrderService {

      @Cacheable("orders")           ← @Retention(RUNTIME), @Target(METHOD)
      @Transactional(readOnly = true)
      public Order findById(long id) { ... }
  }

  ─────────────────────────────────────────────────────────────────────

  COMPILED .class FILE (simplified):

  ClassFile: OrderService
    RuntimeVisibleAnnotations:
      @LService;               ← @Service (RUNTIME → visible)
      @LTransactional;         ← @Transactional (RUNTIME → visible)
        timeout = 30
    Methods:
      findById(J)LOrder;
        RuntimeVisibleAnnotations:
          @LCacheable;
            value = "orders"
          @LTransactional;
            readOnly = true

  ─────────────────────────────────────────────────────────────────────

  RUNTIME REFLECTION FLOW:

  Class<?> clazz = OrderService.class;

  Step 1: Get class-level annotations
          clazz.getAnnotations()
          → [@Service, @Transactional(timeout=30)]

  Step 2: Get specific annotation
          Service svc = clazz.getAnnotation(Service.class);
          → @Service{}

  Step 3: Get method annotations
          Method m = clazz.getDeclaredMethod("findById", long.class);
          m.getAnnotations()
          → [@Cacheable(value="orders"), @Transactional(readOnly=true)]

  Step 4: Read annotation attributes
          Transactional tx = m.getAnnotation(Transactional.class);
          boolean readOnly = tx.readOnly();   → true
          int timeout      = tx.timeout();    → -1 (default)

  ─────────────────────────────────────────────────────────────────────

  SCANNING ALL METHODS FOR AN ANNOTATION:

  for (Method method : clazz.getDeclaredMethods()) {
      Cacheable cache = method.getAnnotation(Cacheable.class);
      if (cache != null) {
          String cacheName = cache.value()[0];
          CacheManager.register(method, cacheName);
      }
  }
```
